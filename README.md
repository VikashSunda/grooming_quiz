# Men’s Grooming Quiz – Lead Magnet

This is a simple interactive quiz designed for a salon/spa to capture leads by offering personalized grooming recommendations.

Users answer a few quick questions about their skin, hair, and lifestyle, and receive suggestions for suitable hairstyles and skincare routines. To view their results, they are asked to enter their name and phone number, along with receiving a ₹300 discount on their first visit.

## Features
- Interactive quiz (5 questions)  
- Personalized recommendations  
- Lead capture (name + phone number)  
- Mobile-friendly design
